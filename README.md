

## Deploy no GitHub Pages
1. Instale dependências:
   ```bash
   npm install
   ```

2. Rode o build:
   ```bash
   npm run build
   ```

3. Publique no GitHub Pages (branch gh-pages):
   ```bash
   npm run deploy
   ```

Certifique-se que no GitHub Pages a fonte está configurada para **branch `gh-pages`**.
